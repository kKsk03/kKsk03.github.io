# 域名购买&配置解析

域名可以分两种方式进行购买，但各有好坏。 

**国内注册商（如阿里、腾讯等）**  
- **好处：**  
冷门域名都很便宜（最低可能不超过10元/年）  
- **坏处：**  
域名解析服务器基本上都是国内的  
若解析对象处于外国网络，有概率后续会被抄水表  
解析服务器IP架设网站等需要国内备案才可通过  

**国外注册商（如Cloudflare）**
- **好处：**
部分注册商提供的功能非常强大（CDN、防御等）  
国内注册商的坏处基本都基本能解决
- **坏处：**
域名比较贵（最低可能也得5美金以上，折合要35元以上）  

## 购买域名

1. 前往`腾讯云`等注册商挑选合适的域名（价格合适就行）购买即可，**需要看清续费价格**，否则后续续费可能会很贵  
2. 购买后，前往其**控制台**即可进行解析  

## 域名解析

解析最好不要使用国内DNS服务器，因为有诸多限制，除非所有服务都走合法渠道，并且需求很大  

1. 注册`Cloudflare`账户，并点击`添加站点`，将已购买的域名填写进去（如：`kksk-blog.site`）  
2. 一路下一步即可，DNS解析界面也直接跳过，推荐打开的按钮全部打开  
3. 在配置`DNS解析服务器页面`时，前往购买域名的注册商处，修改`DNS解析服务器`  
4. 等待其完全接管域名  
5. 前往`DNS --> 记录`处，添加所需要的解析记录即可

::: details 解析帮助
**记录一般用到的：**
|类型|名称|内容|TTL|
|:--:|:--:|:--:|:--:|
| A  |@<br>www|解析到的ipv4地址|默认/300/600|
|CNAME|@<br>www<br>自定义二级域名|解析到的目标域名|默认/300/600|
:::
