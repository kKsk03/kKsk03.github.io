.proto文件转.js / .ts

约 58 字小于 1 分钟

2025-12-07

安装

npm install -g protobufjs-cli

使用

.proto 转 .proto.js

pbjs -t static-module -w commonjs -o <fileName>.proto.js <fileName>.proto

.proto 转 .proto.ts

需要有 .proto.js 作为基础

pbts -o <fileName>.proto.d.ts <fileName>.proto.js
